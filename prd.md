# SUMTEO (Solana) PRD

## Vision
Make reading consistent by turning it into a visible, social growth experience.

## Core Loop
Read -> save a favorite line + 1 sentence -> Reflection NFT -> tree grows -> share.

## Modules
- Reflection input
- Tree DNA generator (offchain)
- Garden renderer (Phaser)
- Team reading + shared tree
- Solana minting (hash + metadata)
- Social sharing

## MVP Scope
- Reflection input and local persistence
- Tree growth visual
- Team creation + weekly goal
- Basic garden sharing
- Minting stub (metadata only)

## Metrics
- Weekly sessions per user
- 4-week retention
- Reflection completion rate
- Team completion rate
